Sobre este projeto

Feito pensando em vocês

Este projeto de TCC foi desenvolvido para apoiar os funcionários sobre o autismo e no acolhimento dos alunos.

A proposta nasceu da necessidade de reunir, em um único espaço, informações simples, práticas e acessíveis sobre os alunos autistas, sem complicações.
